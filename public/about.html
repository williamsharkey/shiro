The ecosystem

One person, seven repositories, a complete browser-native development platform with AI agents that can self-modify their own environment.

shiro core

TypeScript browser OS with Vite build. 78+ built-in commands, npm, git, vi/nano, esbuild-wasm bundler.

TypeScript · ~7,400 LOC

foam

Plain JS sister project. Zero build step — just open index.html. Proves the concept works without a bundler.

JavaScript · ES modules

spirit AI

Claude Code agent library. OSProvider abstraction makes AI agents portable across browser OS environments.

TypeScript · Multi-provider

fluffycoreutils

126 Unix commands in pure TypeScript. FluffyFS interface — 10 methods to abstract any filesystem.

TypeScript · 0 dependencies

nimbus

Multi-agent orchestrator dashboard. Manages concurrent Claude Code workers with live browser previews.

TypeScript · node-pty + xterm.js

windwalker

Puppeteer-based test suite. 10 test levels from boot to self-build. Ensures Foam and Shiro behave identically.

JavaScript · Puppeteer

Remote browser inspection bridge. Execute JS in any browser tab from the command line.
